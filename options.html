Speed control
When you 'long press arrow key', Image viewer will stay at the first/last image before goto other end.
A timer of this period will start counting and continue your action when it stop and not interrupted.
Set debounce period to 0 if you don't want this behavior.
When you 'long press arrow key', Image viewer will display each image approximate this period of time.
The default setting 80ms = 12.5fps (12.5 image per 1 second).
When throttle period was set to be close to 0, fps limit will only depending on you computer performance.
